What do you understand by DOM?
The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web
How to access DOM element?
DOM element can be accessed by methods lik getElementById, getElementsByClassName, getElementsByTagName, querySelector etc.
which method is used for creating new element?
The createElement method is used for creating new element
How to addEventListener?
addEventListener method can be used by specifying the type of event and a callback function
